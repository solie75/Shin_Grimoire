- **컴퓨터 구조 (Computer Architecture)**
    
    - **내용:** 컴퓨터의 하드웨어적 구성 요소와 작동 원리. CPU, 메모리(RAM), 디스크, 캐시, 명령어 집합 등.
        
    - **핵심 질문:** "컴퓨터라는 기계가 물리적으로 어떻게 돌아가는가?"
        
- **운영체제 (Operating System, OS)**
    
    - **내용:** 하드웨어를 관리하는 소프트웨어. 프로세스, 스레드, 스케줄링, 파일 시스템, 가상 메모리 등.
    - **핵심 질문:** "프로그램이 하드웨어 위에서 어떻게 효율적으로 실행되는가?"
        
- **네트워크 (Network)**
    
    - **내용:** 컴퓨터 간의 통신. OSI 7계층, TCP/IP, HTTP, 소켓 통신 등.
    - **핵심 질문:** "데이터가 A 컴퓨터에서 B 컴퓨터로 어떻게 이동하는가?"
        
- **자료구조 및 알고리즘 (Data Structure & Algorithm)**
    
    - **내용:** 데이터를 효율적으로 저장하고 처리하는 방법. 배열, 리스트, 트리, 정렬, 탐색 등.
    - **핵심 질문:** "문제를 해결하기 위해 데이터를 어떻게 다룰 것인가?"
        
- **데이터베이스 (Database)**
    
    - **내용:** 데이터의 저장 및 관리. SQL, 트랜잭션, 인덱싱 등.
