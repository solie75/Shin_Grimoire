- 청구서
	- 과금 정보 및 비용관리 -> 청구 및 결제 -> 청구서 에서 프리 티어 범위를 벗어난 사용이 있는지 확인 가능하다.
- [결제 알람 생성](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/monitor_estimated_charges_with_cloudwatch.html).
	- AWS 계정의 예상 요금 모니터링을 활성화하면, 해당 요금이 계산되어 매일 여러 번 AWS 의 지표 데이터로서 ClundWatch 로 전송된다.
	- Billing and Cost Management -> Billing preferences -> Alert preferences -> Edit
		- Alert preferences -> 'Receive AWS Free Tier Alerts' 와 'CloudWatch Biling Alert'를 체크한다. -> Update 클릭
	- CloudWatch -> Alarms -> All Alarms -> Create Alarm -> Select metric -> Billings $\to$ Total Estimated Charge $\to$ Check 'Use, EstimatedCharges, No alarms' -> Select metric
		- In Configure Actions -> Notification -> Add Notification
			- Select 'in alarm'
			- Select 'Create New topic'
				- input topic name
				- Input E-mail
			- Click 'Next'
		- In Add Alarm details
			- Set 'Alarm name'
			- Write Edit
			- Click Next
		- Click 'Create Alarm'
