# 같은 파일 내의 특정 소제목으로 링크를 거는 법

### 1. 기본 방법 (제목 그대로 링크)

같은 파일 안에서 링크를 걸 때는 **`[[#`** 을 입력하면 됩니다.

1. `[[#` 을 입력합니다.
    
2. 옵시디언이 자동으로 현재 파일에 있는 **소제목 목록**을 팝업으로 보여줍니다.
    
3. 원하는 소제목을 선택하고 엔터(Enter)를 칩니다.
    

**문법:**

Markdown

```
[[#소제목]]
```

---

### 2. 텍스트 변경하여 링크하기 (별칭 사용)

소제목이 너무 길거나, 문맥상 다른 단어로 링크를 걸고 싶다면 **파이프(`|`)** 기호를 사용하여 표시될 텍스트를 바꿀 수 있습니다.

**문법:**

Markdown

```
[[#소제목|화면에 보여줄 텍스트]]
```


# 같은 파일 내에 특정 문단 이나 문장으로 이동하는 링크(Block Link)를 거는 법

### 1. 가장 쉬운 방법 (자동 검색)

링크를 걸고 싶은 위치에서 바로 검색하는 방법입니다.

1. **`[[^`** 를 입력합니다. (대괄호 두 개 + 꺽쇠)
    
2. 현재 파일에 있는 모든 문단들이 팝업으로 뜹니다.
    
3. 링크하고 싶은 문단의 내용을 **일부 입력**하여 검색합니다.
    
4. 원하는 문단을 선택하고 엔터(Enter)를 칩니다.
    

**결과:** 옵시디언이 해당 문단 뒤에 자동으로 고유 ID(예: `^f8e21a`)를 생성하고, 링크에는 `[[#^f8e21a]]` 형태로 입력됩니다.

---

### 2. 수동으로 ID 만들기 (깔끔한 관리)

자동 생성된 무작위 ID가 싫다면, 직접 문단에 이름을 붙여서 링크할 수 있습니다.

1. 링크를 걸고 싶은 문단 **맨 끝**으로 갑니다.
    
2. 한 칸 띄우고 **`^원하는이름`** 을 입력합니다. (예: `^중요내용`)
    
3. 링크를 넣을 곳에서 **`[[#^중요내용]]`** 이라고 입력합니다.
    

**예시:**

> (원본 문단 위치) ...이것은 매우 중요한 결론입니다. **^결론1**
> 
> (링크 거는 위치) 아까 말했던 `    [[#^결론1]]**을 다시 보면...

---

### 3. 별칭 사용하기 (텍스트 변경)

소제목 링크와 마찬가지로 파이프(`|`)를 사용해 링크 텍스트를 바꿀 수 있습니다.

- **입력:** `[[#^결론1|결론 다시 보기]]`
    
- **화면:** `결론 다시 보기` (클릭 시 해당 문단으로 이동)
