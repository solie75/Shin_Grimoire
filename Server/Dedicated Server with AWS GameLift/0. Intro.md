- 소스로부터 언리얼 엔진을 **컴파일**합니다.
- 클라이언트 및 서버 빌드 타겟을 생성하고 게임의 전용 서버 및 클라이언트 빌드를 **패키징**합니다.
- GameLift와 연결하기 위해 **GameLift Server SDK**를 게임에 통합합니다.
- **AWS Command Line Interface (CLI)**로 API 호출을 하는 방법을 배웁니다.
- **자체 컴퓨터와 클라우드 호스팅 가상 머신 모두**에서 전용 서버를 호스팅하고, 그것들이 무엇인지 알게 될 것입니다.
- 클라우드 서비스에 액세스하고 AWS SDK를 사용하기 위해 **AWS Lambda**에서 **서버리스 함수**를 만듭니다.
- **Amazon Cognito**를 사용하여 게임 플레이어가 이메일 인증으로 자신의 로그인 및 비밀번호를 만들 수 있는 **안전한 사용자 풀**을 만듭니다.
- 게임 작업을 수행하기 위해 **사용자 풀 로그인을 통해 부여된 임시 액세스 토큰**을 요구하는 **보안을 구현**합니다.
- **HTTP 요청**을 통해 액세스할 수 있는 엔드포인트가 있는 자체 **Rest API**를 만들고, **Amazon API Gateway**를 사용하여 인증 토큰을 요구하도록 보호합니다.
- **HTTP 요청**이 무엇인지, 그리고 언리얼 엔진 C++ 코드로 **인터넷을 통해 데이터를 전송**하는 데 어떻게 사용하는지 배웁니다. 서버리스 함수를 트리거하고 클라우드 리소스에 액세스하는 방법을 포함합니다.
- HTTP 요청을 위한 **JSON 형식으로 데이터를 포맷**하는 방법과, 클라우드에서 검색된 데이터를 쉽게 사용하기 위해 JSON 데이터를 구조체로 변환하는 언리얼의 **JSON 유틸리티**에 대해 모두 배웁니다.
- 플레이어의 게임 능력치, 커리어, 리더보드 정보를 포함하여 영구적인 게임 데이터를 저장하는 **NoSQL 데이터베이스**를 만드는 방법을 배웁니다. 게임에 완벽한 **고속, 고도로 최적화된 데이터베이스 저장 시스템**인 **DynamoDB 서비스**를 사용합니다.
- 플레이어 이름을 표시하고 **Fast Array Serializers**로 로비 데이터를 효율적으로 복제하는 **전용 서버가 관리하는 게임 로비**를 구현하는 방법을 배웁니다.
- 새 사용자를 위한 **이메일 인증이 포함된 가입 및 로그인 메뉴**와, 커리어 및 리더보드 데이터를 보고 게임 로비를 시작할 수 있는 **게임 액세스 포털**을 만들 것입니다.
- **보안, ID 및 액세스 관리, Identity Center 역할, 정책 및 권한**에 관한 AWS **모범 사례**를 배울 것입니다.
- 모든 AWS API에 대한 **AWS 문서에 액세스**하는 방법과, 클라우드에서 데이터를 액세스, 생성 및 조작할 수 있는 코드를 쉽게 만드는 방법을 배울 것입니다.
- **Amazon GameLift**에서 **게임 세션 및 플레이어 세션**을 만들고 관리할 것입니다.
- 게임의 **클라이언트 빌드**를 만들고 **외부 도구 없이** 클라이언트에서 서버에 연결할 수 있습니다. **그냥 게임을 실행하고, 로그인하고, 게임 내에서 연결**하면 됩니다.
- 게임 서버에 연결하지 않고도 **클라우드 서비스에 연결**하는 방법을 배울 것입니다. 즉, **싱글 플레이어 게임**에서도 가능합니다.
- **AWS 결제 및 관리 콘솔**을 면밀히 모니터링하고 **결제 알람**을 설정하는 방법을 배우며, 이 외에도 훨씬 더 많은 것을 배울 것입니다.
